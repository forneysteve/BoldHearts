# BoldHearts  
A grounded, masculine, contemplative website for spiritual direction, pastoral counseling, and leadership soul care.

BoldHearts exists to offer a steady, confidential space for people carrying emotional, spiritual, or vocational weight. The site is designed to feel like a mountain retreat for the soul — structured, warm, and quietly courageous.

---

## 🌄 Purpose  
BoldHearts provides:
- Spiritual direction  
- Pastoral counseling  
- Leadership soul care  
- A grounded, unhurried space for honest conversation and discernment  

The website reflects this ethos through masculine design elements: stone textures, earth tones, strong typography, and clean structure.

---

## 📁 Project Structure

```
/
├── index.html
├── about.html
├── services.html
├── contact.html
├── testimonials.html
├── styles.css
└── assets/ (optional images or media)
```

Each page is fully responsive and built with clean HTML + CSS (no frameworks required).

---

## 🎨 Design Language  
**Masculine, grounded, contemplative.**

- **Colors:** charcoal, deep earth brown, slate plum, sage, rust, sandstone  
- **Typography:** Cormorant Garamond (headings), Inter (body)  
- **Imagery:** mountains, stone, wood, natural textures  
- **Layout:** structured grids, strong lines, generous spacing  

---

## 🚀 Deployment  
This site can be deployed using:
- **GitHub Pages** (recommended for simplicity)  
- **Firebase Hosting**  
- Any static hosting provider  

If using GitHub Pages:
1. Go to **Settings → Pages**  
2. Set **Source** to “Deploy from a branch”  
3. Select `main` and `/ (root)`  
4. Save  

Your site will publish automatically.
